Melecio Silva

Software Engineer

Hi! My name is Melecio.

I am a software engineering student in Thinkful's Engineering program. Before transitioning to tech, I studied and worked in the healthcare industry as an aspiring physical therapist.

Outside of traveling, I have spent my entire life in the Old Dominion, Virginia. I currently reside in Richmond, but I spent my college years living in the Valley while attending James Madison University and Mary Baldwin University.

I was introduced to programming in early 2020, and have had a strong interest in learning more ever since. Months passed and I arrived at a crossroads - my pursuit to becoming a physical therapist suddenly came to a halt. Fortunately, this gave me permission to reflect on my experiences, reassess my goals and aspirations, and pursue programming with a purpose. My hope is that I can leverage the years of education and service I gained in the healthcare industry to benefit my community on a larger scale using software.

About Me

My professional background and academic development are in strength & conditioning and rehabilitation. I have also worked in a diversity of industries ranging from construction to customer service. I am passionate about how I can leverage my own unique skill set and experiences to maximize my efficacy as a software engineer. Portfolio

Local Library

This project makes retrieving library information on accounts, books, and authors more accessible. I used several JavaScript array methods to extract pertinent data from objects including the following: find(), filter(), map(), indexOf(), join(), reduce(), and more. More importantly, I learned how to effectively access nested data and how to refactor my code using object shorthand, spread operators, and destructuring techniques.

Decoder Ring

This project allows users to encode or decode text using Caesar's Shift, Polybius Square, and Substitution ciphers. I used Test Driven Development with JavaScript to produce code, develop unit tests with Mocha and Chai, and handle errors. I also learned how to use regular expressions and developed proficiency writing modular code for helper functions.

Rock, Paper, Scissors

This game features a user interface that allows a user to make a game selection for each round of play. I used HTML and CSS to develop the layout and style of the webpage, which included the use of flexbox and button pseduoclasses. I used JavaScript methods to select DOM elements, modify elements, and respond to events. Building this project gave me first-hand experience manipulating the DOM in order to create a more dynamic and responsive user experience.

Virginia Breweries

This project was designed to help users navigate and access pertinent information on Virginia's breweries. It also features an option for users to search breweries by name or address to filter their results. While developing this project, I used axios to send HTTP requests to the Open Brewey DB and used async/await to handle the responses. In addition, I used Twitter Bootstrap to create the layout, styling, and responsive design of the webpage.
